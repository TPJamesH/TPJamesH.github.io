One of the past projects I worked on

Embedded System: Operating System and Interfacing, Semester 2, 2024

This video demonstrates the concept of bare metal programming – writing firmware which directly runs on the hardware without having any underlying abstraction, through the implementation of an embedded Bare OS (Operating System) on Raspberry PI 4. The requirements for the OS consist of a CLI, terminal formatting through ANSI code, UART modification, and mailbox peripheral. Furthermore, there will be demonstration of the implementation of more complex features (e.g. CLI input, “help” and “help…” commands, terminal formatting, command history, autocomplete, and UART modification).
